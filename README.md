## 📌 Objetivo
Desarrollar una aplicación móvil que permita a los usuarios gestionar:
- Dispositivos conectados al bastón
- Cuidadores asignados
- Ubicación y zonas seguras
- Botón de emergencia

## 🛠️ Tecnologías

- Dart
- SQLite (encriptado)
